\# Kohli Store API



Production-style grocery delivery backend built with NestJS, Prisma, and PostgreSQL.



\## Features



\- REST API backend

\- PostgreSQL database

\- Prisma ORM

\- Product catalog

\- Categories

\- Cart

\- Orders

\- Coupons

\- Notifications

\- Static image hosting

\- Render deployment



\## Tech Stack



\- NestJS

\- TypeScript

\- Prisma

\- PostgreSQL

\- Render
